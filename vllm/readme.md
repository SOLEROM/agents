# vllm

* https://docs.vllm.ai/en/latest/
* a high-performance LLM inference engine/server used to serve models at scale.


Its focus is maximizing throughput and GPU efficiency when serving models to many users or requests. 


```
    vLLM → infrastructure for AI services

    Ollama → local LLM runtime



    Ollama → better for local experiments and small agents

    vLLM → useful if you want one GPU serving many clients or containers
```
